# Hera Yacht Charter - Versão CNPJ 65.078.945/0001-71

Esta pasta reúne uma variação do site da **Hera Yacht Charter** com foco em charter internacional de iates, vinculada ao CNPJ **65.078.945/0001-71**.

## Visão Geral

O conteúdo foi adaptado para comunicar a consultoria em destinos como Mediterrâneo, Caribe e outros roteiros exóticos, mantendo o padrão visual náutico da marca.

-   **Tecnologias:** HTML5, CSS3, JavaScript (ES6+)
-   **Design:** Responsivo, luxuoso e adequado a campanhas globais.
-   **Uso típico:** Landing page para clientes brasileiros interessados em charter internacional.
